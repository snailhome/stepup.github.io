Linux Mint 安装后要做的几件事

蜗牛

 2010-01-01 08:03:00
目录: System
标签: Linux Mint 更新源 输入法 语言包

Linux Mint 1.72 是一个粉美观的桌面 Linux 系统，网评还不错的桌面，试用看看感觉还蛮好的。 但是要记得安装好系统后要记得先做好几件事，才能用得舒心。

修改更新源

系统一装完，先修改更新源在做更新，不然会等吐血 打开菜单 -> 系统管理 -> 软件源 -> 官方仓库项 分别选择镜像里的主要和基础两个选项，找最快的源，我选择的是：

＊ 主要：http://mirrors.hustunique.com/linuxmint ＊ 基础：http://mirrors.yun-idc.com/ubuntu

然后点击更新缓存，然后打开终端开始更新吧

sudo apt-get clean && sudo apt-get update && sudo apt-get upgrade -y

修改语言支持

打开菜单 -> 首选项 -> 语言设置窗口 －> 输入法页 -> 点击添加/删除语言，我只保留了中英美三国语言，语言、地区都选中国，点击应用到整个系统

添加输入法

linux mint支持多种输入法框架，这方面做得还是比较好的。打开菜单 -> 首选项 -> 语言设置窗口 －> 输入法页 -> 点击添加对xxx的支持按钮，就可以添加相应输入法的框架了

我使用小企鹅fcitx输入法，在输入法页点击添加对fcitx的支持按钮，就会开始安装进程，完成后不要点击安装可选部件件，不然会一股脑给你安装了61个可选部件，我勒个去～～，那怎么做？ 关闭语言设置窗口，打开菜单->系统管理->新立得软件包管理器，过滤fcitx，然后在列表中分别找到fcitx-googlepinyin，fcitx-table-wubi，fcitx-table，在列表项上右键选择标志以便安装，然后点击应用`就开始安装了。 这里安装的是谷哥拼音和五笔。

fcitx输入法界面比较美观，汉化也比较全面 ibus也安装起来用过，但界面没那么好，汉化也不全，另外还有一个不足，在General页的Show property panel:项应选择Do not show，不然任务栏经常会被卡住

现在应用注销一下以使输入法生效，重新登录后就可以在任务栏上看到键盘图标了 现在就可以用： Ctrl+空格：开关输入法 Ctrl+Shift：切换输入法

添加Firefox的语言包

打开Firefox，Open menu -> Add-ones 搜索chinese，安装 Chinese Simplified(zh-CN) Language Pack 另外还可以选择安装Firebug、BaiduSearch 关闭Firefox，再重新打开，界面就是中文的了
